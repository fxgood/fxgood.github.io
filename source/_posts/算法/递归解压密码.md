遇到过好几次类似的解压字符串的题目，每次写起来都比较困难，今天看到一种不错的思路，加以记录。

<!--more-->

[外星密码](https://www.luogu.com.cn/problem/P1928)

```cpp
#include <iostream>

using namespace std;

string decode() {
    int k;
    char ch;
    string s = "", str = ""; //s是最终答案,str是被压缩的数字
    while (cin >> ch) {
        if (ch == '[') { //找到了被压缩的字符串
            cin >> k;
            str = decode();   //递归
            while (k--)
                s += str;
        } else if (ch == ']')    //如果找到了待压缩字符串的末尾
            return s;
        else   //如果没有被压缩
            s += ch;  //直接添上这个字符
    }
}

int main() {
    cout << decode();
    return 0;
}
```

知识点：虽然数字和英文字母连在一起，但是cin还是能够把它抠出来（scanf也能做到），惊，我以前竟然都不知道。
