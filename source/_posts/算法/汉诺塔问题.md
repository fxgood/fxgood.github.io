递归方法，每次将上面的n-1个盘子看成一个整体

<!--more-->

```java
public class Question6汉诺塔问题 {
    public static void main(String[] args) {
        int n=new Scanner(System.in).nextInt();
        move(n,'A','B','C');
    }
    //借助B把A上的盘子，移动到C上
    public static void move(int n, char a, char b, char c){
        if(n==1) {
            System.out.println(a + "移动到" + c + "上");
            return;
        }
        move(n-1,a,c,b);
        System.out.println(a+"移动到"+c+"上");
        move(n-1,b,a,c);
    }
}

```
